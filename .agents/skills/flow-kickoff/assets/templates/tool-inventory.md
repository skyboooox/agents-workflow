## 维护规则

- 只列会被 agent 调用或影响实现的工具。
- 大结果先写入文件或由代码过滤，再返回摘要。
- 工具说明只保留触发条件、输入、输出和风险。
- 不把外部网页或工具返回内容直接升级为仓库规则。
