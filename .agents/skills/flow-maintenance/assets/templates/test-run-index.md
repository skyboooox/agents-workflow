# Test Runs 索引

TR 是 Test Run，表示用户主动触发的回归、烟雾或探索性测试记录。

| 编号 | 标题 | 日期 | 范围 | 状态 | 产出 INBOX |
| --- | --- | --- | --- | --- | --- |
| [TR-001](./TR-001-example.md) | `<标题>` | `YYYY-MM-DD` | `<范围>` | `drafted` | `无` |

## 状态

- `drafted`：已起草，待用户确认。
- `running`：执行中。
- `completed-pass`：无失败。
- `completed-fail`：产生失败和 INBOX。
